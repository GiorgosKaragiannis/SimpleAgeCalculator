# AgeCalculator.js
A simple Age Calculator built with HTML, CSS, and JavaScript. Users pick their birthdate with flatpickr, and Luxon calculates their age in years and months. Extra days are shown separately. The app checks for a valid date and displays everything in a clean, centered layout.
For more detailed context on the requirements and objectives of this project, you can refer to the original source: https://roadmap.sh/projects/age-calculator.
